# Armazenamento Azure

    Contas de armazenamento são o tipo de armazenamento de base no Azure.
    O armazenamento oferece um repositório de objetos escalonável para objetos de dados 
    e serviços de sistema de arquivos na nuvem.

 ## Suas quatro opções de configuração:

 - Blob do Azure: Texto e dados binários. *Quando usar? Se você precisar provisionar um armazernamento de dados que 
guardará, mas não consultará dados, a opção mais econônica é configurar 
uma conta de armazenamento Blobs. Funciona bem com imagens e dados não
estruturados.* 

 - Arquivos do Azure: Compartilhamento de arquivos gerenciados para implantação
local e em nuvem.

 - Fila: repositório de mensagens confiáveis entre 
componentes do aplicativo.

 - Tabela: Um repositório NoSql para armazenamento sem esquemas
de dados estruturados.

    **Pode-se usar o Armazenamento do Azure como a base para o armazenamenyto
    ao peovisionar uma tecnologia de plataforma de dados,
    como Azure Data Lake Storage e o HDInsight, ou também para uso autônomo.**

## Segurança de dados:

O Armazenamento do Azure criptografa todos os dados gravados
nele. Também oferece um controle refinado sobre quem tem acesso aos 
seus dados.
